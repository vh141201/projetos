Esse projeto foi feito a parte para me ajudar durante minha segunda iniciação científica na faculdade, onde precisava escolher os sensores 
com melhor desempenho ou os que não saiam do padrão. Para agilizar e facilitar a escolha de sensores foi feito esse programa, que com a ajuda
de ferramentas de visualização de dados como MDS e T-SNE consegue, logo após a medida ser realizada, mostrar onde se dá o sensor com relação aos outros.
A imagem a) que está neste diretório ilustra um pouco da interface do programa, e a imagem b) ilustra um exemplo de uso com várias medidas de sensores.
Cada cor é um tipo de sensor diferente (que está em outra pasta). A idéia é, se o sensor sai muito do cluster central de sensores, ele deve ser descartado.
Isso poupa tempo na hora de escolher os sensores ideais ou descartar os defeituosos.
